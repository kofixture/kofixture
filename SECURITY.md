# Security Policy

## Supported Versions

| Version | Supported |
|---|---|
| 0.x-SNAPSHOT | ✅ Best effort |

## Reporting a Vulnerability

Please **do not** open a public GitHub issue for security vulnerabilities.

Instead, report via [GitHub Private Security Advisory](https://github.com/kofixture/kofixture/security/advisories/new).

We will respond on a best-effort basis and coordinate a fix and disclosure timeline with you.
